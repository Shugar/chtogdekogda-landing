# chtogdekogda-landing

We use:
  - nodejs
  - npm
  - bower
  - brunch

How to install: 
```sh
npm install
bower install
```

How to start server:
```sh
brunch w -s
```

How to build:

```sh
brunch build
```

Guidelines for writing your code: 


##Stylus syntax


####Вложенность

```sh
.example-1
  .example-2
    .example-2-1
  .example-3
    .example-3-1
```

####Блоки с контентом
